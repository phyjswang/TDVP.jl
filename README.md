# TDVP.jl

Calculate Loschmidt rate function for dynamical quantum phase transitions ([PRL 110, 135704](https://link.aps.org/doi/10.1103/PhysRevLett.110.135704)) of transverse field Ising chain in the thermodynamical limit using TDVP algorithm ([SciPost Phys. Lect. Notes 7](https://scipost.org/SciPostPhysLectNotes.7)).
